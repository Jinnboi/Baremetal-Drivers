# Baremetal-Drivers
A collection of lightweight, baremetal drivers for the STM32F411RET6 MCU

## Project Structure
### UART
#### UART_RX
Implementation and test logic for receiving serial data from terminal

#### UART_TX
Implementation and test logic for transmitting serial data to terminal

#### UART_RXTX
Implementation and test logic for receiving serial data, then transmitting serial data to terminal
